## Usage

```tsx
import ProfileCard from "@/components/ProfileCard";
```

```tsx
<ProfileCard
  name="Juan Sánchez Vinuesa"
  title="Telematic Engineer"
  handle="Jsanchezv2201"
  status="Online"
  contactText="Contact Me"
  avatarUrl="/avatar.jpg"
  showUserInfo
  enableTilt={true}
  enableMobileTilt={false}
  behindGlowColor="rgba(125, 190, 255, 0.67)"
  behindGlowEnabled
  innerGradient="linear-gradient(145deg,#60496e8c 0%,#71C4FF44 100%)"
  // Optional dimension props
  cardAspectRatio={1} // 1 = square, 16/9 = landscape, etc.
  cardWidth="min(80vw, 400px)" // responsive max width
/>
```

## Dimension Props

| Prop              | Type      | Description                                              |
| ----------------- | --------- | -------------------------------------------------------- |
| `cardAspectRatio` | `number`  | Aspect ratio (width/height). Default: `0.718` (portrait) |
| `cardWidth`       | `string`  | CSS width. Enables landscape mode when set               |
| `cardMaxWidth`    | `string`  | Additional `max-width` CSS value                         |
| `cardMaxHeight`   | `string`  | Additional `max-height` CSS value                        |
| `showDetails`     | `boolean` | Show/hide name and title overlay. Default: `true`        |
